# VS Code使用

现在真的是逢人就推荐vs code



1、打开的编译器

如果你是单击代码文件，其实是浏览，再点击其他的会覆盖掉，双击才是打开新的tab，你可以观察单击时显示的文件名是斜体，而双击时才是正体

如果要修改，设置的名为：`workbench.editor.enablePreview:false`
